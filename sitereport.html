Learning throughout the project

Throughout this web development project, I learned many foundational skills in HTML and CSS. I started this project by building a structure of the web using simple HTML elements while visualizing the final result the base was written with elements like nav, header, footer, section and many more. I planned to implement a consistent navigation menu bar linking all pages making it easier for user to move through sites. After making the skeleton, I used CSS elements to make the web pages more stylish and less complex. During this, I learned about CSS Grid and CSS Flexbox, for instance, in the project page I used grid layout to arrange the mutiple project cards.The navigation menu uses Flexbox to distribute links evenly.
