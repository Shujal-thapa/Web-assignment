This website was developed as part of the CSY1063 Web Development module. The main aim of this project was to learn the fundamentals of HTML and CSS while building a multi-page responsive website...

Semantic HTML elements such as header, nav, main, section, article and footer were used to improve structure and accessibility...

CSS Grid was implemented on the projects page to demonstrate modern layout techniques. Separate CSS files were used for each page to keep styling organised and beginner-friendly...

Throughout this assignment, version control using Git and GitHub was used to track progress and manage daily commits. This helped in understanding real-world development workflows.
